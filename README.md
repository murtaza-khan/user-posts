
## Description

Implented User post CURD API's for create user and create post and show the list. I have used NestJS framework with TypeScript;

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# for run with docker
$ docker-compose up -d --build
```
